## Instalações backend:
new terminal
cd backend
npm install
npm  mysql2
npm run dev


## Instalações frontend:
new terminal
cd frontend
npm install
npm run dev 

Deverá aparecer local:  http://localhost:XXXX/

Extras:
npm install react-icons (ícons react)
npm install react-markdown (formatação de markdown)
